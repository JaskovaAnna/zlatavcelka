# Zlatá Včelka – Prezentační web v HTML/CSS/JS

## Popis projektu
Tento projekt je statický prezentační web pro fiktivní rodinné včelařství "Zlatá Včelka". Web byl vytvořen jako cvičný projekt zaměřený na moderní front-end techniky bez použití frameworků.

## Funkcionalita
- Responzivní layout přizpůsobený mobilním zařízením i desktopům.
- Plynulé scroll animace sekcí při načítání stránky.
- Produktové modální okno s detailními informacemi po kliknutí na kartu.
- Mobilní hamburgerové menu s animací.
- Kontaktní formulář s demo odesláním.
- Aktivní zvýraznění navigačních odkazů při scrollování.
- Open Graph meta tagy pro sdílení na sociálních sítích.

## Jak začít
1. Naklonujte tento repozitář:
   ```bash
   git clone https://github.com/JaskovaAnna/zlatavcelka.git
   ```
2. Otevřete `index.html` v prohlížeči.

## Požadavky
- Moderní webový prohlížeč s podporou JavaScriptu.
- Připojení k internetu pro načtení Google Fonts.
